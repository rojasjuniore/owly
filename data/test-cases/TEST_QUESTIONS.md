## 5.2 ❌ No Califica (DSCR)
**Pregunta:**
> Investor quiere comprar rental property de $400,000, renta de mercado $1,800/mes, PITIA estimado $3,200/mes. Credit score 660, 20% down. ¿DSCR loan?

**Respuesta Esperada:**
> ❌ **NO CALIFICA** para DSCR Loan.
> 
> **Razón de rechazo:**
> - DSCR Ratio: $1,800 / $3,200 = **0.56** ❌
> - Mínimo requerido: 0.75-1.0+ dependiendo del lender
> 
> **DSCR = Rent / PITIA**
> - DSCR ≥ 1.0: Cash flow positivo (ideal)
> - DSCR 0.75-0.99: Algunos lenders aceptan con más down
> - DSCR < 0.75: Generalmente rechazado
> 
> **Opciones:**
> 1. Buscar propiedad con mejor rent/price ratio
> 2. Poner más down payment para reducir PITIA
> 3. Bank Statement program si tiene income personal fuerte
> 
> **Confianza:** 95%

## Notas para Implementación

### Formato de Respuesta Esperado
```
[RESULTADO]: ✅ SÍ CALIFICA / ❌ NO CALIFICA / ⚠️ CON CONDICIONES
[PROGRAMA]: Tipo de préstamo
[ANÁLISIS]: Lista de criterios evaluados

[FUENTES]: PDFs/Guidelines consultados
[ALTERNATIVAS]: Si no califica, qué otras opciones
```

### Cálculo de Confianza (Code-Based)
```python
confidence = base_score
for criterion in criteria:
    if criterion.status == "PASS":
        confidence += criterion.weight
    elif criterion.status == "BORDERLINE":
        confidence += criterion.weight * 0.5
    elif criterion.status == "FAIL":
        confidence -= criterion.weight * 2
return min(max(confidence, 0), 100)
```
